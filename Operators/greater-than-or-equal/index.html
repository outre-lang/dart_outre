    Greater Than Or Equal¶

    Greater Than Or Equal (>=) can be used to find if a is greater than or equal to b.

    Syntax
    operand1 >= operand2
    Example
    a >= b

1 >= 2
# false

3 >= 3
# true

4 >= 3
# true
