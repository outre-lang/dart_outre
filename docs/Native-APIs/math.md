# Math

## `Math.random`

Returns a random value within `0` and `1`.

```title="Signature"
-> : Number
```

```title="Example"
value := Math.random();

# prints 0.3245124
print(value);
```

## `Math.pi`

Returns the value of PI.

```title="Signature"
-> : Number
```

```title="Example"
pi := Math.pi();

# prints 3.1415926535897932
print(pi);
```

## `Math.sin`

Returns sine of the value.

```title="Signature"
-> Number value : Number
```

```title="Example"
value := Math.sin(1);

# prints 0.8414709848078965
print(value);
```

## `Math.min`

Returns minimum value in `a` and `b`.

```title="Signature"
-> Number a, Number b : Number
```
