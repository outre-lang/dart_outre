# Globals

## `typeof`

Takes in a value and returns the type. The type will be any one of `Boolean`, `Function`, `NativeFunction`, `AsyncNativeFunction`, `Null`, `Number`, `String`, `Object`, `List` or `Future`.

```title="Signature"
-> Any value : String
```

```title="Example"
# prints "Number"
print typeof(10);

# prints "String"
print typeof("Hello");
```
